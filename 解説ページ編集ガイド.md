# 解説ページ編集ガイド

## ファイル構成

- **normal_reimu_a.html**: Normal難度 霊夢
  - 6つのステージがあり、各ステージにタイムスタンプがある
  - ステージボタンで切り替え可能

- **extra_reimu_a.html**: Extra難度 霊夢
  - 単一のステージ
  - タイムスタンプで異なるシーンに移動

## 動画ID の設定

### 動画IDの取得方法

YouTubeのURLから動画IDを取得します：
- URL例: `https://www.youtube.com/watch?v=dQw4w9WgXcQ`
- **動画ID**: `dQw4w9WgXcQ`（`v=`の後ろの部分）

### 動画IDの指定場所

HTMLの`<iframe>`タグの`data-video-id`属性に指定します：

```html
<iframe id="videoFrame" data-video-id="dQw4w9WgXcQ" ...></iframe>
```

**normal_reimu_a.html の現在の状態:**
```html
<iframe id="videoFrame" data-video-id="" ...></iframe>
```
→ 空なので、動画IDを入れてください

**extra_reimu_a.html の例:**
```html
<iframe id="videoFrame" data-video-id="ujmtniAMzek" ...></iframe>
```
→ 既に動画IDが指定されています

## 編集方法

### normal_reimu_a.html の編集

#### 時間を変更する場所

JavaScriptの `stageData` オブジェクト内のみ（**1箇所**）

```javascript
const stageData = {
    1: {
        timestamps: [
            { time: 10, label: 'タイムスタンプ1' },
            { time: 30, label: 'タイムスタンプ2' }
        ]
    },
    2: {
        timestamps: [
            { time: 20, label: 'タイムスタンプ3' },
            { time: 40, label: 'タイムスタンプ4' }
        ]
    },
    // ... 以降ステージ3～6も同様
};
```

**変更する場合の例:**
- ステージ1のタイムスタンプ1を45秒にしたい → `{ time: 45, label: 'タイムスタンプ1' }`

#### 解説文を変更する場所

HTMLの `content-panel` セクション内のコンテンツ要素（複数箇所）

**IDの対応関係:**
- ステージ N のタイムスタンプ K（配列の K 番目） → `content-{N}-{K}`
- ステージ1のタイムスタンプ1 → `content-1-0`
- ステージ1のタイムスタンプ2 → `content-1-1`
- ステージ2のタイムスタンプ1 → `content-2-0`

```html
<div id="content-{stage}-{index}" class="stamp-content">
    <h3>タイトル</h3>
    <p>ここに解説文を書きます。</p>
</div>
```

**変更する場合の例:**
```html
<div id="content-1-0" class="stamp-content">
    <h3>タイムスタンプ1</h3>
    <p>ステージ1の序盤について解説します。</p>
</div>
```

### extra_reimu_a.html の編集

#### 時間を変更する場所

JavaScriptの `timestamps` 配列内のみ（**1箇所**）

```javascript
const timestamps = [
    { time: 47, label: '弾消し' },
    { time: 216, label: 'クランベリートラップ' },
    { time: 60, label: '中盤' },
    { time: 120, label: 'ボス戦' }
];
```

**変更する場合の例:**
- 「弾消し」のタイムスタンプを50秒にしたい → `{ time: 50, label: '弾消し' }`

#### 解説文を変更する場所

HTMLの `content-panel` セクション内のコンテンツ要素（複数箇所）

**IDの対応関係:**
- `timestamps` 配列の K 番目のタイムスタンプ → `content-{K}`
- 1番目（弾消し） → `content-0`
- 2番目（クランベリートラップ） → `content-1`
- 3番目（中盤） → `content-2`
- 4番目（ボス戦） → `content-3`

```html
<div id="content-{index}" class="stamp-content">
    <h3>タイトル</h3>
    <p>ここに解説文を書きます。</p>
</div>
```

**変更する場合の例:**
```html
<div id="content-0" class="stamp-content">
    <h3>弾消し</h3>
    <p>弾消しのテクニックについて詳しく解説します。</p>
</div>
```

## ポイント

- **時間編集は1箇所のみ**: `stageData` または `timestamps` の `time` プロパティだけ変更
- **解説文**: 対応するコンテンツ要素の `<p>` タグ内に記入
- **タイムスタンプの追加**: 配列に新しいオブジェクトを追加し、対応するコンテンツ要素も追加
- **タイムスタンプのラベル変更**: 配列の `label` プロパティを変更
- **アクセス時の動作**: 
  - normal_reimu_a は 1面のタイムスタンプが表示され、コンテンツは未選択
  - extra_reimu_a はタイムスタンプが表示され、コンテンツは未選択
